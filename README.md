How to use:
Simply run the file, follow any given instructions, and use the program as you wish. If you experience problems, please read the section below.

Troubleshooting:
- Try downloading the latest version of .NET Framework from Microsoft: https://dotnet.microsoft.com/en-us/download/dotnet-framework
- Try downloading the latest version of .NET from Microsoft: https://dotnet.microsoft.com/en-us/download
- Try downloading the latest version of Visual C++ Redistributable from Microsoft: https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist
- WARNING: Only follow this step if you are confident in what you are doing. If you are still experiencing problems, as a last resort, you could try disabling your antivirus. In some cases, even uninstalling it may be necessary, as some antivirus programs may still block the program even when turned off.
